# color-scheme-5-nebula-spark (dark scheme)

Dark theme derived from the firm's purple/deep blue direction. 
Designed for data-dense financial dashboards and analytics portals.

Anchor colors from the team's existing direction, extended with 
semantic tokens for charts, status indicators, and data tables.

## Usage

CSS: import `tokens.css` and use custom properties.
Python/JS: import `tokens.json` for programmatic access.
Preview: open `preview.html` in a browser.
