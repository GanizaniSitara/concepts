# Colour Schemes

Five framework-agnostic CSS colour schemes. Each lives in its own subfolder with tokens (CSS custom properties), optional JSON, and base styles.

| # | Name | Type | Folder |
|---|---|---|---|
| 1 | Sapphire Dusk | Light | [color-scheme-1-sapphire-dusk](color-scheme-1-sapphire-dusk/) |
| 2 | Moss Tidal | Light | [color-scheme-2-moss-tidal](color-scheme-2-moss-tidal/) |
| 3 | Slate Citrus | Light | [color-scheme-3-slate-citrus](color-scheme-3-slate-citrus/) |
| 4 | Orchid Sunset | Light | [color-scheme-4-orchid-sunset](color-scheme-4-orchid-sunset/) |
| 5 | Nebula Spark | Dark | [color-scheme-5-nebula-spark](color-scheme-5-nebula-spark/) |

Schemes 3 and 4 are overlay themes that layer on top of Scheme 1 (Sapphire Dusk).
