# Colour Schemes

Five framework-agnostic CSS colour schemes.

| # | Name | Type | Folder |
|---|---|---|---|
| 1 | Moss Tidal | Light | [color-scheme-1-moss-tidal](color-scheme-1-moss-tidal/) |
| 2 | Slate Citrus | Light | [color-scheme-2-slate-citrus](color-scheme-2-slate-citrus/) |
| 3 | Orchid Sunset | Light | [color-scheme-3-orchid-sunset](color-scheme-3-orchid-sunset/) |
| 4 | Nebula Spark | Light | [color-scheme-4-nebula-spark](color-scheme-4-nebula-spark/) |
| 5 | Nebula Dark | Dark | [color-scheme-5-nebula-dark](color-scheme-5-nebula-dark/) |

Schemes 2 and 3 are overlay themes that layer on top of Scheme 1 (Moss Tidal), inheriting its full extended palette. Schemes 4 and 5 share the same Nebula colour family in light and dark variants.
