# Color Scheme 1 — Cobalt Reef (light)

Light theme built on deep blues, cyans and teals. Large palette with extensive pastel tints.

## Files

| File | Purpose |
|---|---|
| `tokens.css` | All design tokens as CSS custom properties |
| `tokens.json` | Same tokens in JSON (for programmatic access) |

## Usage

```html
<link rel="stylesheet" href="tokens.css">
```
