# Color Scheme 4 — Nebula Spark (dark)

Dark theme derived from a purple/deep-blue direction. Designed for data-dense financial dashboards and analytics portals.

## Files

| File | Purpose |
|---|---|
| `tokens.css` | All design tokens as CSS custom properties |
| `tokens.json` | Same tokens in JSON (for programmatic access) |

## Usage

```html
<link rel="stylesheet" href="tokens.css">
```

Or in JS/Python, import `tokens.json`.
