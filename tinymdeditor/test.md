# Welcome to TinyMD

A **fast** Markdown editor for Windows.

## Features

- Live preview as you type
- Dark theme
- Ctrl+S to save
- Tab key works in the editor

## Code Example

```go
func main() {
    fmt.Println("Hello, TinyMD!")
}
```
